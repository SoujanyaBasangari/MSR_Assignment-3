This package contains the following:

1. The full evaluation dataset of 7,998 repository URLs.
2. The sample of 400 repositories used for testing the name extraction module.
3. The sample of 4,094 pairs of paper references and repositories used for testing the reference extraction module.
4. The sample of 160 repositories used for testing the dataset extraction module.
5. The sample of 252 repositories from model zoos used to test the framework extraction module.
6. The validation set of information from 978 repositories used to test the domain inference module.
7. The sample of 80 repositories used to evaluate the system as a whole.

With the exception of #1, each of the evaluation samples also includes results from our evaluation.
